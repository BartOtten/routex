    Routex.Extension.Interpolation.NonUniqError exception (Routex v1.3.2)

    Copy Markdown View Source

    Raised when a list of routes contains routes with the same path and verb.

    [%Route{
  path: "/foo"
  verb: :get},
%Route{
  path: "/foo"
  verb: :post}, # <-- different
%Route{
  path: "/foo"
  verb: :get} # <-- duplicate
]

    Solution: use a combination of interpolated attributes that form a unique set.
